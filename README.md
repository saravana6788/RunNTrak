RunNTrak is a fitness app that records the Run times and provide a report of your fitness level

Below are the teck stack used to develop this app

* Kotlin
* Jetpack Compose
* Multi-Module
* CLEAN Architecture
* Work Manager
* Dynamic Feature Install
